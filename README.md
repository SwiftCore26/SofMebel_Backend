## Loyihani o‘rnatish

```bash
# Virtual environment yaratish
python3 -m venv .venv

# Virtual environment ni yoqish
source .venv/bin/activate

# uv paket menejerini o‘rnatish
pip install uv

# pyproject.toml dagi barcha kutubxonalarni o‘rnatish
uv sync

# kutubhona o'rnatish
uv add (kutubxona nomi)
```
